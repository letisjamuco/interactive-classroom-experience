# D1.0 - Project Description

## Project Title
**Interactive Classroom Experience**

## Course
Interactive Systems (M110)

## Developer
Letisja Muco

## Platform
Unity 3D

## Type
First-Person 3D Environment

## Concept
The *Interactive Classroom Experience* is a small 3D environment based on a classroom.  
The player can move around the room and interact with a few simple objects such as a laptop, notebook, headphones, and a light switch.  
Each object reacts to the player’s actions with small changes like sound, light, or a short message.

## Goal
To create a simple Unity project that includes basic movement, object interaction, lighting control, and ambient sound.  
The project aims to show an understanding of Unity’s main features in a familiar setting.
